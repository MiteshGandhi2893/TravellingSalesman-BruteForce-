Steps to be followed for compilation and Execution of TSPProblem.java

Step 1 : Please copy the file TSPProblem.java in your directory along with makefile.txt and input.txt from mgandhi2_P1.tar.gz file

Step 2 : It is important that there exist an input.txt file in the same directory where TSPProblem.java,makefile.txt is kept.if it          is not there then please create one with name input.txt

Step 3 : Type command make -f makefile.txt



The step 3 will  compile the code ,Execute the code and creates and open a txt file named TSPOutput.txt in the same directory
